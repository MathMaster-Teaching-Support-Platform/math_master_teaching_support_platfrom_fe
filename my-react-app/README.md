## 💻 Available Scripts

In the project directory, you can run the following standard commands:

### `npm run dev`
Runs the app in the development mode.
It will typically open [http://localhost:5173](http://localhost:5173) to view it in your browser. The page will reload when you make changes.

### `npm run build`
Builds the app for production to the `dist` folder.
It correctly bundles React in production mode and optimizes the build for the best performance.

### `npm run preview`
Bootstraps a local web server that serves the production build in `dist/`. Suitable for testing the production build locally.

### `npm run lint`
Runs ESLint to analyze the code and find potential issues.
